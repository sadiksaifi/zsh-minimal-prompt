### Installation

> Using [ZAP](https://www.zapzsh.com/)

```bash
plug "sadiksaifi/zsh-minimal-prompt"
```

### Manual Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/sadiksaifi/zsh-minimal-prompt.git ~/.zsh-minimal-prompt
   ```
2. Add the following to your `~/.zshrc` file:
   ```bash
   source ~/.zsh-minimal-prompt/zsh-minimal-prompt.zsh-theme
   ```
